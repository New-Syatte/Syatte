## 🖐️Introduce
이 프로젝트는 Syatt가 고객들에게 제품과 교육을 판매하고, 해당 주문과 일정을 관리할 수 있는 사이트입니다. 포페인팅 관련 물품의 구매 / 배송 등의 E-Commerce 서비스 와 오프라인 교육 예약 및 일정 관리 시스템을 제공합니다.
